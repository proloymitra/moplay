Quick Help

System Requirements

  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • JavaScript enabled
  • Stable internet connection
  • Minimum 1GB RAM recommended

Supported Browsers

  • Google Chrome (recommended)
  • Mozilla Firefox
  • Safari (macOS/iOS)
  • Microsoft Edge

Keyboard Shortcuts

  • FEnter/Exit fullscreen
  • RRefresh current game
  • ESCExit fullscreen
  • CTRL+FSearch games

Account Features

  • Save game progress
  • Track achievements
  • Compete on leaderboards
  • Join community chat

Common Issues & Solutions

Game won't load or shows black screen

  • Refresh the page (F5 or Ctrl+R)
  • Clear your browser cache and cookies
  • Disable browser extensions/add-ons
  • Try a different browser
  • Check your internet connection

Account login issues

  • Double-check your email and password
  • Use the "Forgot Password" feature
  • Clear browser cache and try again
  • Contact support if issues persist

Games running slowly or laggy

  • Close other browser tabs and applications
  • Check your internet speed
  • Try lowering game quality settings
  • Restart your browser
  • Update your browser to the latest version

Audio not working in games

  • Check if browser audio is muted
  • Look for audio controls within the game
  • Check system volume settings
  • Try refreshing the page
  • Test audio on another website
